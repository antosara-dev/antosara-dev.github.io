**antosāra:** *quintessence; the essence of a thing in its purest.*

My name is Vinay Penmatsa. I have over 20 years of experience designing and developing software products, libraries and services.

At *Salesforce*, I worked on building and running a secrets management service based on the open-source Hashi Corp Vault. Using Golang, Kubernetes, Terraform, Docker, Spinnaker, AWS services like DynamoDB, I have contributed to the security, performance and scalability of the service as well as the DevOps.

At *SAP Ariba*, I have built several products in the Enterprise Sourcing and Supplier Relationship areas. I have led cross-team and cross-product security initiatives like export control compliance, encryption in motion and at rest while building common platform security libraries for stronger encryption with advanced security protocols. I have built full stack applications and tools for Supplier Risk management that included supplier matching with Duns and Bradstreet Org data using Apache Lucene, data de-duplication with machine learning and similarity-based clustering in Python, integration with external services like GDACS and Geo names using batch job execution based on Java platform, Spring framework, microservices architecture.

I have built dynamic web service composition and execution framework that enabled customers to integrate their ERP, SRM systems with SAP/Ariba Enterprise Sourcing using Java, Apache CXF, REST, SOAP while contributing to Apache CXF security open source.

In my free time, I coach the Blitz Bots First Tech Challenge ([FTC 25750](https://ftcscout.org/teams/25750)) team helping students learn software and robotics.

If you like to use my skills and experience in building software, please reach out to me at [vinay@antosara.com](mailto:vinay@antosara.com).
